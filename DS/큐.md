큐란 선입선출(FIFO) 특성을 가진 자료구조로 여기서 선입선출은 가장 먼저 들어온 데이터가 가장 먼저 나간다는 뜻이다

큐에 삽입/삭제 속도는 O(1)이다

C++ 코드로는 이렇게

~~~
queue<int> q;

q.push(0);
q.push(1);
q.push(2);
q.push(3);

cout << "size: " << q.size() << "\n";

while (!q.empty()) {
    cout << q.front();
    q.pop();
}
~~~

Python 코드로는 이렇게 구현할 수 있다

~~~
from collections import deque

q = deque()

q.append(0)
q.append(1)
q.append(2)
q.append(3)

print(f"size: {len(q)}")

while q:
    print(q.popleft())
~~~
